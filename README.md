## quick start
### prerequisites
- [Clarinet](https://github.com/hirosystems/clarinet) - Stacks smart contract development tool
- [Node.js](https://nodejs.org/) v16+ - JavaScript runtime
- [Git](https://git-scm.com/) - Version control

```bash
# Clone the repository
git clone https://github.com/your-username/banking-regulatory-technology-contract.git
cd banking-regulatory-technology-contract

# Install dependencies
yarn install

# Verify installation
stx balance

# Run contract checks
stx balance

# Run tests
yarn test

# Build for production
yarn run build
```
